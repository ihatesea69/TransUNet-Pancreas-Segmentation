Architecture

Three-Stage Pipeline

CNN Encoder: ResNet-style backbone extracts hierarchical features at multiple scales (1/4, 1/8, 1/16, 1/32 resolution).

Transformer Bottleneck: 12-layer Vision Transformer captures global context through self-attention mechanisms.

CNN Decoder: U-Net architecture with skip connections progressively recovers spatial resolution for precise segmentation.

Medical Segmentation Decathlon

Task07: Pancreas

High-quality annotated dataset from Memorial Sloan Kettering Cancer Center. Includes 420 portal venous phase CT scans with expert-labeled pancreas and tumor segmentations.

Automatically downloaded via MONAI in Notebook 01. Preprocessing includes orientation standardization (RAS), isotropic 1.0mm resampling, and HU windowing [-175, 250] for soft tissue focus.

  • Volumes 420 scans
  • Modality CT (Portal Venous)
  • Labels Background, Pancreas, Tumor
  • Format NIfTI (.nii.gz)
  • Size ~11.4GB compressed
  • Split 80% / 10% / 10%
  • License CC-BY-SA 4.0

Get Started

Three ways to use TransUNet: Interactive Notebooks, CLI, or Python API

1

Clone Repository

Get the code and setup environment

git clone https://github.com/ihatesea69/TransUNet-Pancreas-Segmentation.git cd TransUNet-Pancreas-Segmentation uv sync
2

Run Notebooks

Interactive exploration and training

jupyter notebook 01_Data_Exploration_and_Processing.ipynb
3

Train Model

Start training with CLI or notebooks

python main.py train --variant small --epochs 50
4

Run Inference

Segment new CT volumes

python main.py inference --checkpoint model.pth --input scan.nii.gz
